## Overview

The 1tap.video API allows you to integrate video recording, editing, and sharing capabilities into your applications. Our RESTful API provides endpoints for managing videos, users, and analytics.

## Authentication

All API requests require authentication using an API key. Include your API key in the request header:

```http
Authorization: Bearer YOUR_API_KEY
```

<Warning>
Keep your API key secure and never expose it in client-side code. Use environment variables to store your API key.
</Warning>

## Base URL

All API endpoints are relative to the base URL:

```
https://api.1tap.video/v1
```

## Rate Limits

API requests are rate limited to ensure fair usage:

- **Free Plan**: 100 requests per hour
- **Pro Plan**: 1,000 requests per hour
- **Enterprise**: Custom limits

Rate limit headers are included in all responses:

```http
X-RateLimit-Limit: 1000
X-RateLimit-Remaining: 999
X-RateLimit-Reset: 1640995200
```

## Error Handling

The API uses standard HTTP status codes and returns error details in JSON format:

```json
{
  "error": {
    "code": "INVALID_REQUEST",
    "message": "The request is missing required parameters",
    "details": {
      "field": "title",
      "reason": "required"
    }
  }
}
```

## Videos

### Create Video

Create a new video recording session.

<ParamField path="POST /videos" body="application/json">
  <Param name="title" type="string" required>
    Title of the video
  </Param>
  <Param name="description" type="string">
    Optional description
  </Param>
  <Param name="settings" type="object">
    Recording settings
    <Param name="resolution" type="string" default="1080p">
      Recording resolution (720p, 1080p, 4K)
    </Param>
    <Param name="fps" type="integer" default="30">
      Frame rate (30, 60, 120)
    </Param>
    <Param name="audio" type="boolean" default="true">
      Include audio recording
    </Param>
  </Param>
</ParamField>

<ResponseField name="201">
  <Param name="id" type="string">
    Unique video identifier
  </Param>
  <Param name="title" type="string">
    Video title
  </Param>
  <Param name="status" type="string">
    Recording status (pending, recording, completed, failed)
  </Param>
  <Param name="created_at" type="string">
    ISO 8601 timestamp
  </Param>
  <Param name="recording_url" type="string">
    URL for starting the recording
  </Param>
</ResponseField>

## Webhooks

Configure webhooks to receive real-time notifications about video events.

### Create Webhook

<ParamField path="POST /webhooks" body="application/json">
  <Param name="url" type="string" required>
    Webhook endpoint URL
  </Param>
  <Param name="events" type="array" required>
    Array of event types to subscribe to
  </Param>
  <Param name="secret" type="string">
    Webhook secret for signature verification
  </Param>
</ParamField>

### Supported Events

- `video.created` - Video recording started
- `video.completed` - Video processing completed
- `video.failed` - Video processing failed
- `video.shared` - Video shared via link
- `video.viewed` - Video viewed by someone

### Webhook Payload

```json
{
  "id": "evt_1234567890",
  "type": "video.completed",
  "created": "2023-01-01T12:00:00Z",
  "data": {
    "video": {
      "id": "vid_1234567890",
      "title": "My Video",
      "status": "completed"
    }
  }
}
```

## Code Examples

### JavaScript Example

```javascript
import { OneTapVideo } from '@1tap-video/sdk';

const client = new OneTapVideo({
  apiKey: process.env.ONETAP_API_KEY
});

// Create a new video
const video = await client.videos.create({
  title: 'My Tutorial',
  description: 'A step-by-step guide',
  settings: {
    resolution: '1080p',
    fps: 60,
    audio: true
  }
});

console.log('Video created:', video.id);

// Start recording
await client.videos.startRecording(video.id);

// Stop recording after 30 seconds
setTimeout(async () => {
  await client.videos.stopRecording(video.id);
  
  // Create share link
  const share = await client.shares.create(video.id, {
    expires_at: '2024-12-31T23:59:59Z',
    allow_download: true
  });
  
  console.log('Share URL:', share.url);
}, 30000);
```

### Python Example

```python
from onetap_video import OneTapVideo

client = OneTapVideo(api_key='your_api_key')

# Create a new video
video = client.videos.create(
    title='My Tutorial',
    description='A step-by-step guide',
    settings={
        'resolution': '1080p',
        'fps': 60,
        'audio': True
    }
)

print(f'Video created: {video.id}')

# Start recording
client.videos.start_recording(video.id)

# Stop recording after 30 seconds
import time
time.sleep(30)
client.videos.stop_recording(video.id)

# Create share link
share = client.shares.create(video.id, {
    'expires_at': '2024-12-31T23:59:59Z',
    'allow_download': True
})

print(f'Share URL: {share.url}')
```
